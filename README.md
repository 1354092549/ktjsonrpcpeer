## KtJsonRpcPeer
**KtJsonRpcPeer** is a Kotlin library that implements [JSON-RPC 2.0](https://www.jsonrpc.org/specification) in Peer mode.  
It's full-duplex, supporting two-way procedure call.  
It can work well with WebSocket.
